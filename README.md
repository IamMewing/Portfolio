# Dharma Teja Portfolio
A modern, interactive developer portfolio showcasing my projects, skills, and experience in web development and Generative AI. Built with React, Tailwind CSS, Framer Motion, and GSAP, the portfolio features a premium skeuomorphic design, smooth animations, and a responsive user experience.

## Features

- Modern Skeuomorphic UI
- Fully Responsive Design
- Smooth Page Animations
- GSAP & Framer Motion Effects
- Interactive Project Showcase
- AI Portfolio Assistant (Neo)
- Dynamic Skills Section
- GitHub & Resume Integration
- Contact Form
- Optimized Performance

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- GSAP
- React Icons
- JavaScript
